# 🚀 My Flask API Project

## 📌 Introduction
This project is a Flask-based REST API with full documentation support using Sphinx and Flasgger.

It demonstrates how to build a well-structured backend system with clear API contracts and automated documentation.

---

## 🧩 Features
- RESTful API built with Flask
- Interactive API documentation (Swagger UI)
- Auto-generated code documentation (Sphinx)
- Modular project structure using Blueprint

---

## 🛠 Tech Stack
- Python 3
- Flask
- Flasgger (OpenAPI / Swagger)
- Sphinx (Documentation generator)
